# /retain <create|retrieve>

Save the context of the conversation concisely for future conversation
post-clear.

## Usage

- `/retain create` - Create RETAIN.md based on the current conversation
- `/retain retrieve` - Retrive and understand content of RETAIN.md

## Actions

- create: Write current context into RETAIN.md, containing file structure and a
  brief summary of the purposes of the domain and file
- retrieve: Ingest the content of RETAIN.md into the current conversation

## Purpose

Enables frequent clears while preserving the context of conversations.
